# GI Endoscopy Technician Training Program

A self-contained, offline-capable PWA for training GI Endoscopy Technicians from zero prior knowledge to job-ready competency.

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
# Commits /docs folder → GitHub Pages serves it automatically
```
